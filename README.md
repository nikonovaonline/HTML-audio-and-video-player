# HTML Audio and Video Player

## Project Overview

This is a lightweight HTML media player built to practice HTML5 `<audio>` and `<video>` elements. The project demonstrates fundamental skills in web development, cross-browser compatibility, and project documentation. It is part of my ongoing portfolio showcasing practical web development skills.

## Key Features

* Play audio and video files directly in the browser
* Support for multiple sources via `<source>` tags
* Built-in controls: play, pause, volume, and progress
* Simple and clean layout suitable for embedding in websites
* Cross-browser compatibility ensured via correct MIME type usage

## Technologies Used

* HTML5
* CSS for optional styling
* Git and GitHub for version control and deployment

## Purpose and Learning Outcomes

* Solidify knowledge of HTML5 audio and video elements
* Ensure media works across different browsers and devices
* Practice creating clear project documentation for professional presentation
* Strengthen portfolio for international web development opportunities

## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/nikonovaonline/HTML-audio-and-video-player.git
   ```
2. Open `index.html` in any modern browser.
3. Select and play audio or video using the standard controls.
